net.openstack.Providers.Rackspace.Objects Namespace
The net.openstack.Providers.Rackspace.Objects namespaces define the object model for communicating with Rackspace services over REST APIs.
Classes
  ClassDescription
ArchiveFormat
Represents a archive format for the Cloud Files Extract Archive operation.
BulkDeletionFailedObject
Describes an object which could not be deleted by a bulk deletion operation, along with a status providing the reason why the deletion failed.
BulkDeletionResults
Represents the detailed results of a bulk deletion operation.
Domain
Represents the domain for a user account.
RackspaceCloudIdentity
Extends the CloudIdentity class by adding support for specifying a Domain for the identity.
