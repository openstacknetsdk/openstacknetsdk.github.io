net.openstack.Providers.Hp Namespace
The net.openstack.Providers.Hp namespaces provide an implementation of the core OpenStack interfaces for accessing HP cloud products and services, as well as additional HP-specific functionality.
Classes
  ClassDescription
HpIdentityProvider
Provides an implementation of IIdentityProvider for operating with HP's Cloud Identity product. This provider supports authentication using a username/password combination or an access key/secret key combinatiton, and supports scoped tokens when credentials are represented with CloudIdentityWithProject.
PredefinedHpIdentityEndpoints
Provides endpoints for the HP Cloud Identity Service.
